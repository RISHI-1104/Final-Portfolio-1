# Rishikesh S — Portfolio (innotech1)

Minimal static scaffold for a dual-theme landing + portfolio.

Contents:
- `index.html` — main page with Landing, Hero, About, Learning, Projects, Strengths, Blog, Contact, Footer
- `styles.css` — theme, layout, animations
- `script.js` — typing intro, morph, theme toggle, smooth scroll, form handling
- `assets/profile.svg` — placeholder profile image

Quick preview (open `index.html` in a browser).

Replace `assets/profile.svg` with a real photo when ready. Theme preference saved in `localStorage`.
